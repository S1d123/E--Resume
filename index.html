Library Management System


Our Library management System is a graphical user interface which is employed for automating a library. The purpose of the system is to allow for storing details of a large number of books, magazines, Journals, thesis and allow to add, search, borrow, return facilities separately to Administrator/Librarian, staff and students. Earlier there was a problem of lack of Security, lack of data storage, lack of Information, Issuing of books was slow, slow retrieval of data.   • MySQL

All details will be available in a click for user and librarian.
